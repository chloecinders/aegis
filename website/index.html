Ouroboros Moderation

A Discord moderation bot built with Rust. High performance, zero bloat, and completely open-source. Dedicated to doing exactly one thing.

Modern Semantics

Infers arguments intuitively using replies. Reply to someone or an automod log and the bot fills in the rest instantly.

Advanced Logging

Pulls crucial additional data from Discord's audit log, giving you deeper insights like identifying who deleted a specific message.

Dynamic Caching

Giant cache sizes exactly where it matters while keeping memory consumption low. Supports huge active channels effortlessly.

Zero Bloat

No leveling, no economy, no music. Expect this bot to not turn into a kitchen sink. Strictly built for top-tier moderation.
